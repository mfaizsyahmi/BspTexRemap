INTRODUCTION
------------
This package (DPG_LP) uses pyleri to define a domain-specific language (DSL) for laying out DearPyGui (DPG) items, allowing you to build DPG layouts using a markup file.

The current DSL supports the following DPG items:
- group
- text
- image
- collapsible headers
- horizontal rules
- DPG tree node
- tables
- text links
- radio
- combobox
- listbox

REQUIREMENTS
------------
DearPyGui 1.10.0
pyleri    1.4.2
